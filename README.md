# CostaTech - Tienda de Periféricos Gaming

Proyecto eccomer para el módulo de diseño de interfaces web.

**Autor:** Hamza Lahrour Charif

## Funcionalidades de la web

Para esta tarea he desarrollado toda la lógica con JavaScript, aplicando las siguientes funciones:

- **Carrusel:** Deslizador de productos en la página principal.
- **Botón de subir:** Botón dinámico en la esquina inferior que aparece al hacer scroll.
- **Acordeón:** Menú desplegable para organizar la información técnica del producto.
- **Pestañas:** Sistema para alternar de forma dinámica entre el mapa y el vídeo de presentación.
- **Notificaciones:** Avisos visuales al realizar acciones, como copiar el código de descuento.
- **Carrito persistente:** Panel lateral que guarda los productos añadidos usando `localStorage` para que no se borren al recargar la página.
- **Accesibilidad:** Uso de HTML semántico y buenas prácticas para conseguir la máxima nota en auditorías.
